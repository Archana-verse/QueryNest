# QueryNest

QueryNest is an AI-powered assistant with support for:
- 💬 AI Chat (Groq API)
- 📄 PDF Search (FastAPI + Qdrant)
- 🌐 Web Search (SerpAPI)
- ☁️ Weather Info (wttr.in)

## Tech Stack

- **Frontend**: React + Vite
- **Backend**: FastAPI
- **LLM**: Groq API (Mixtral / LLaMA 3)
- **Vector DB**: Qdrant
- **PDF Parsing**: PyPDF2
- **Voice Input**: Web Speech API
